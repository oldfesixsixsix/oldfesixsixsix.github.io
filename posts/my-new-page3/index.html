Page3 - Hugo 新增搜尋頁

September 18, 2025

1. 搜尋引擎#

預設 PaperMod 是用 Lunr.js。

它會去讀 /index.json 來跑全文檢索。

index.json 的生成

你需要在 config.toml 裡啟用 JSON 輸出：

[outputs]
  home = ["HTML", "RSS", "JSON"]

這樣 Hugo build 的時候才會輸出 public/index.json。

本地測試

hugo server -D

http://localhost:1313/index.json 要能打開，看到文章 JSON。

2. PaperMod 搜尋機制#

內建支援 Lunr.js 搜尋 但記得要打開 params 裡的 enableSearch

你在 config.toml（或 hugo.toml）裡開啟：

[params]
  enableSearch = true

[params] 是 主題自訂參數。

去讀取 /index.json，然後在前端跑全文檢索。

3. 查看內建的 JS#

PaperMod 的搜尋功能不是靠 Hugo 自己魔法出現的，而是靠 主題本身內建的 JS + 模板。

ls themes/PaperMod/assets/js/

新一點的 PaperMod repo 是 fastsearch.js

這版 PaperMod 有搜尋頁 (search layout)，但不是「右上角放大鏡」的 popup 模式，而是 「跳轉到 /search 頁面」 的設計。

也就是說：

fastsearch.js 會在 /search/ 頁面啟動，從 index.json 拉全文索引。

cat themes/PaperMod/layouts/partials/header.html | grep search

header.html 裡的這段：

$is_search := eq (site.GetPage .KeyName).Layout `search`
<a href="{{ .URL | absLangURL }}" ...>

表示導航列如果配置了一個「Search」menu item，它就會連到 /search。

4. 新增搜尋頁#

在 content/search/_index.md 建立搜尋頁：

hugo new search/_index.md
Content "/mnt/d/myblog/content/search/_index.md" created

編輯內容

---
title: "Search"
layout: "search"
---

在 config.toml（或 hugo.toml）加一個 menu 指向 /search：

[[menu.main]]
identifier = "search"
name = "Search"
url = "/search/"
weight = 10

重新 hugo server -D → 你會在導覽列看到「Search」連結，點進去就是全文檢索頁。
