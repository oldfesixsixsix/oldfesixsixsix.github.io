September 19, 2025

1. 先按照 lovable clone to github 自帶的 README.md 操作#

Follow these steps:

# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev

npm install ncp --save-dev

3. 一鍵打包#

複製到指定路徑下

npm run deploy

package.json

 "scripts": {
    "dev": "vite",
    "build": "vite build",
    "build:dev": "vite build --mode development",
    "lint": "eslint .",
    "preview": "vite preview",
    "deploy": "ncp dist D:/workspace1/username/repo1"
  },

複製到指定路徑下 並且將該路徑下檔案推送

"scripts": {
    "dev": "vite",
    "build": "vite build",
    "build:dev": "vite build --mode development",
    "lint": "eslint .",
    "preview": "vite preview",
    "deploy": "ncp dist D:/workspace1/username/repo1 
	&& cd D:/workspace1/username/repo1 
	&& git add . 
	&& git commit -m \"Update repo1 build\" 
	&& git push"
  },

commit message error

error: pathspec 'repo1' did not match any file(s) known to git
error: pathspec 'build'' did not match any file(s) known to git

這代表 commit 訊息有問題，記得是用 "將訊息包起來
